# Banner Grabber

A Python-based banner grabbing tool built to learn networking and cybersecurity fundamentals.

## Features

- TCP Banner Grabbing
- SSH Detection
- HTTP Detection
- FTP Detection
- SMTP Detection
- Hostname Resolution
- Port Validation
- Modular Architecture

## Project Structure

banner.py
scanner.py
parser.py
utils.py
