# 01 - Ollama API

My first project using Ollama.

## Features

- Basic API request
- Interactive chatbot
- Streaming responses

## Files

app.py
- Simple API call

chat.py
- Interactive chatbot

stream.py
- Streaming chatbot

requirements.txt
- Python dependencies

## Installation

```bash
pip install -r requirements.txt
```

Run Ollama

```bash
ollama serve
```

Run

```bash
python app.py
```
