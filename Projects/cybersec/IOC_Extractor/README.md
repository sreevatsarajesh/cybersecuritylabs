# IOC Extractor

A Python-based IOC (Indicator of Compromise) extraction tool built to learn cybersecurity, threat hunting, and regular expressions by identifying malicious indicators from text files.

## Features

- IPv4 Address Extraction
- Domain Extraction
- URL Extraction
- Email Extraction
- MD5 Hash Detection
- SHA1 Hash Detection
- SHA256 Hash Detection
- CVE ID Detection
- Duplicate Removal
- JSON Export
- Modular Architecture
